# Piedrazul Application Models (Hexagonal)

Status: Not implemented yet.

## Objective

Define models for the core domain and explicit port contracts.

## Required Model Layers

- Domain core: entities, value objects, domain services
- Application core: use cases and orchestration rules
- Port contracts: inbound and outbound interfaces
- Adapter contracts: mapping rules between external and domain models

## Output Artifacts

- Core domain model catalog
- Port contract definitions
- Adapter mapping specification
- Requirement-to-model traceability map
